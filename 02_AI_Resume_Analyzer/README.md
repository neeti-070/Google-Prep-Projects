# AI Resume Analyzer

A Streamlit web application that analyzes PDF resumes and extracts key information like skills, experience, and education.

## Features
- PDF resume upload
- Skill detection
- Experience extraction
- Education information parsing
- Clean web interface

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Download spaCy model: `python -m spacy download en_core_web_sm`
3. Run the app: `streamlit run app.py`

## Usage
1. Upload a PDF resume
2. View extracted skills, experience, and education
3. Get instant analysis results
